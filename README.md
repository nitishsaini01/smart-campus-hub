# Smart Campus Hub

Smart Campus Hub is a **web-based platform designed to help students and teachers share academic resources and collaborate efficiently** within a college environment.

The platform allows users to upload study materials, participate in discussions, and stay updated with important announcements. The goal is to create a centralized space where academic resources and communication are easily accessible.

---

## Features

* User authentication for students and teachers
* Upload and access academic resources (notes, PDFs, links)
* Discussion forum for asking questions and collaboration
* Announcements section for important updates
* Organized resource management by subjects or topics
* Simple and responsive user interface

## Installation

Clone the repository and install dependencies.

```
git clone https://github.com/yourusername/smart-campus-hub.git
cd smart-campus-hub
npm install
```

---

## Running the Project

Start the backend server:

```
node server.js
```

Open the frontend in your browser to access the platform.
